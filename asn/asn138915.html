Live BGP Lookup

This tool allows you to lookup current live BGP routing data for a specific autonomous system.
This will report the current IPv4/IPv6 prefixes announced from or via a specific AS number.

Routers around the internet announce the IP prefixes they handle routing for to their peers.
Each peer decides, for each IP prefix, which path is chosen for the active routing table entry.
Generally, a more specific IP prefix will be chosen over a less specific one but this is not always the case.
Each border router decides what next-hop will be taken for each prefix.

NOTE: By default, this tool will report 32-bit ASNs in ASDOT format, use the checkbox to use ASPLAIN.
